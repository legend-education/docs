# What does Mathematical Practice 5 (MP5) mean?

<div class="answer-box">
  `MP5` says students should pick the right tool for the math at hand. The tool might be a ruler, a number line, a graphing calculator, a spreadsheet, a piece of dynamic geometry software, or pencil and paper. The point is that the student chooses the tool based on what the problem needs, knows what the tool can and cannot do, and uses it well enough to get useful information.
</div>

## Official text

Mathematically proficient students consider the available tools when solving a mathematical problem. These tools might include pencil and paper, concrete models, a ruler, a protractor, a calculator, a spreadsheet, a computer algebra system, a statistical package, or dynamic geometry software. Proficient students are sufficiently familiar with tools appropriate for their grade or course to make sound decisions about when each of these tools might be helpful, recognizing both the insight to be gained and their limitations. For example, mathematically proficient high school students analyze graphs of functions and solutions generated using a graphing calculator. They detect possible errors by strategically using estimation and other mathematical knowledge. When making mathematical models, they know that technology can enable them to visualize the results of varying assumptions, explore consequences, and compare predictions with data. Mathematically proficient students at various grade levels are able to identify relevant external mathematical resources, such as digital content located on a website, and use them to pose or solve problems. They are able to use technological tools to explore and deepen their understanding of concepts.

## What students do

- Pick a tool because it fits the problem, not because it is on the desk.
- Use the tool well: read a ruler at the right end, read a graphing window correctly, set up a spreadsheet without typos.
- Use estimation to catch errors the tool might mask.
- Know when a tool will not help. Sometimes paper and pencil is fastest.

| Grade band | What `MP5` looks like |
| --- | --- |
| K-2 | Choosing between counters, ten-frames, a number path, and fingers based on the problem. |
| 3-5 | Choosing among rulers, protractors, base-ten blocks, fraction strips, and arrays. |
| 6-8 | Using calculators for messy arithmetic, spreadsheets for tables of values, and dynamic geometry tools for exploration. |
| 9-12 | Choosing between a graphing calculator, a computer algebra system, a spreadsheet, and pencil and paper based on what the problem rewards. |

## Classroom application

`MP5` shows up the moment students have access to more than one tool. Make the choice visible. A useful prompt at the start of a problem is, "What tools could help here, and which one are you choosing?" At the end, "Would a different tool have been faster or clearer?"

`MP5` is also about knowing when not to use a tool. A student who reaches for the calculator to compute 9 x 5 is not yet using tools strategically. A student who notices the prices in a spreadsheet should be summed and uses `SUM` rather than adding by hand is.

Rubric language for `MP5`:

- **Approaching:** Student uses whichever tool is at hand, regardless of fit.
- **Meets:** Student selects a tool that fits the problem and uses it correctly.
- **Exceeds:** Student justifies the tool choice and uses estimation or a second tool to check the result.

## Common teacher questions

<AccordionGroup>
  <Accordion title="Does using a calculator help or hurt MP5?">
    Both, depending on how it is used. A calculator used for messy arithmetic that would otherwise eat the lesson supports `MP5`. A calculator used to skip estimation and reasoning works against it. The teacher move is to require an estimate before the calculator press and a reasonableness check after.
  </Accordion>

  <Accordion title="What tools should be available at each grade?">
    The standards do not prescribe specific tools, but a working list looks like this: K-2 uses physical counters, ten-frames, number lines, and rulers; grades 3-5 add fraction strips, arrays, area models, and protractors; grades 6-8 add calculators, spreadsheets, and dynamic geometry software; grades 9-12 add graphing calculators or equivalent software and statistical packages. The point is access plus instruction on when each tool is the right one.
  </Accordion>
</AccordionGroup>

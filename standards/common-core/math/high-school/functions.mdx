# What are the Common Core high school Functions standards?

<div class="answer-box">
  Functions in high school is a conceptual category with four domains: Interpreting Functions (`F-IF`), Building Functions (`F-BF`), Linear, Quadratic, and Exponential Models (`F-LE`), and Trigonometric Functions (`F-TF`). Students use function notation, analyze key features of graphs and tables, build new functions from existing ones, distinguish situations modeled by linear, quadratic, and exponential functions, and (for advanced tracks) work with trigonometric functions of real-number arguments.
</div>

## What it covers

- Interpreting Functions: understand the concept of a function and use function notation; interpret functions that arise in applications; analyze functions using different representations (`F-IF`).
- Building Functions: build a function that models a relationship between two quantities; build new functions from existing functions (`F-BF`).
- Linear, Quadratic, and Exponential Models: construct and compare linear and exponential models; interpret expressions for functions in terms of the situation they model (`F-LE`).
- Trigonometric Functions: extend the domain of trigonometric functions; model periodic phenomena; prove trigonometric identities (`F-TF`).

## Classroom application

`F-IF.A` formalizes the function concept students met in grade 8 (`8.F.A`). The notation f(x) is new in high school and merits explicit teaching: f(x) is the output of f at input x. The most common student error is treating f(x) as f times x. A useful corrective is repeated practice reading f(3) as "the output of f when the input is 3."

`F-IF.B.4` is the high-leverage standard for graph interpretation. Given a graph of bottle-water-volume versus time, students name intercepts (initial value), intervals of increase or decrease, maximums, and end behavior, all in the units of the context. Connecting symbolic features (zeros of a quadratic) to graphical features (x-intercepts) to contextual features (when the rocket lands) is the central work.

`F-IF.B.6` brings rate of change forward as a function property, not just a line property. The average rate of change of f(x) = x squared from x = 1 to x = 3 is (f(3) - f(1))/(3 - 1) = (9 - 1)/2 = 4. For nonlinear functions, the rate depends on the interval; this anticipates calculus.

`F-BF.B.3` is the transformations cluster: vertical shifts (f(x) + k), vertical stretches and reflections (k f(x)), horizontal stretches and reflections (f(kx)), horizontal shifts (f(x + k)). Desmos or any graphing tool with sliders is the canonical instructional tool here. Students manipulate sliders and observe the geometric effect.

`F-LE.A.1` distinguishes linear from exponential growth. The proof in `F-LE.A.1.a` is the conceptual capstone: linear functions add a fixed amount per unit interval; exponential functions multiply by a fixed factor per unit interval. A useful classroom task is a table comparing $100 plus $10 per year (linear) versus $100 at 10 percent compound interest (exponential) over 30 years. The exponential explodes; the linear stays manageable.

`F-LE.A.4` connects exponentials and logarithms operationally. Solving 2 to the x = 10 requires expressing x as log base 2 of 10. The standard restricts to bases 2, 10, and e for required work; advanced students extend further.

`F-TF` is mostly plus-standards beyond the required `F-TF.A.1`, `F-TF.A.2`, `F-TF.B.5`, and `F-TF.C.8`. Radian measure (`F-TF.A.1`) is the most-skipped foundational idea: the radian measure of an angle is the length of the arc that the angle cuts out on the unit circle. From there, sine and cosine can be defined for any real number using the unit circle, not just acute angles in right triangles.

## Common teacher questions

<AccordionGroup>
  <Accordion title="Which standards are required for every student?">
    All non-plus standards in `F-IF`, `F-BF.A.1`, `F-BF.A.2`, `F-BF.B.3`, `F-BF.B.4.a`, `F-LE`, and `F-TF.A.1`, `F-TF.A.2`, `F-TF.B.5`, `F-TF.C.8` are required. Standards marked (+) are for students preparing for advanced courses.
  </Accordion>
  <Accordion title="Where does logarithms work live?">
    The required logarithm work is `F-LE.A.4` (solving exponential equations using logarithms). The full inverse relationship between exponents and logarithms (`F-BF.B.5`) is a plus-standard, though it typically appears in Algebra 2.
  </Accordion>
  <Accordion title="Should I teach transformations in Algebra or Functions?">
    Transformations (`F-BF.B.3`) are categorized under Functions because they apply to function families generally. In a traditional sequence, basic transformations show up alongside each new function family (quadratics in Algebra 1, exponentials and others in Algebra 2). The Functions framing emphasizes the common structure across families.
  </Accordion>
</AccordionGroup>

<div class="canonical-summary">
  The Common Core high school Functions conceptual category covers four domains. In Interpreting Functions (`F-IF`), students use function notation, interpret key features of graphs and tables, calculate average rate of change, and analyze functions in different representations. In Building Functions (`F-BF`), they construct functions from contexts, work with arithmetic and geometric sequences, transform functions, and find inverses. In Linear, Quadratic, and Exponential Models (`F-LE`), they distinguish and construct linear and exponential models, observe that exponentials eventually exceed polynomials, and solve exponential equations using logarithms. In Trigonometric Functions (`F-TF`), they extend trig functions to all real numbers using the unit circle, model periodic phenomena, and prove the Pythagorean identity.

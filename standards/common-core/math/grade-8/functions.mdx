# What are the Common Core Functions standards for Grade 8?

<div class="answer-box">
  Functions in grade 8 (`8.F`) is a new domain. Students learn that a function is a rule that assigns exactly one output to each input (`8.F.A.1`), compare functions represented as equations, tables, graphs, or descriptions (`8.F.A.2`), recognize linear and nonlinear functions (`8.F.A.3`), construct a function to model a linear relationship (`8.F.B.4`), and describe qualitatively the relationship between two quantities by sketching or interpreting a graph (`8.F.B.5`). This is the formal beginning of function as a mathematical object.
</div>

## What it covers

- Define, evaluate, and compare functions (`8.F.A`).
- Use functions to model relationships between quantities (`8.F.B`).

## Classroom application

The function concept in `8.F.A.1` is best introduced as a vending-machine metaphor: each input (button press) produces exactly one output (item). If pressing the same button could give two different items, it would not be a function. A useful prompt is to show a table or graph and ask whether it could be a function. A table where x = 2 maps to both y = 4 and y = 5 is not a function. A graph that fails the vertical line test (a circle, for instance) is not a function. The standard is explicit that function notation (f(x)) is not required in grade 8; the focus is on the concept and on rate-of-change comparisons.

`8.F.A.2` is the cluster's classroom workhorse. Pose two linear functions in two different representations: one as a graph, one as a table, or one as a verbal description and another as an equation. Ask which has the greater rate of change, or which has the greater y-intercept. The work forces students to extract slope and intercept from each representation and compare.

`8.F.A.3` formalizes the distinction between linear and nonlinear. A graph with a constant slope (a straight line) is linear; anything curved is nonlinear. Area-versus-side-length for a square (A = s squared) is the canonical nonlinear example: the points (1,1), (2,4), (3,9) do not lie on a straight line. Volume-versus-edge-length for a cube (V = s cubed) is another.

`8.F.B.4` is the construction standard. Given any of (description, two points, table, graph), students extract the rate of change (slope) and initial value (y-intercept), then write y = mx + b. The interpretive piece is essential: in a phone plan that charges $20 setup plus $5 per month, the 20 is the y-intercept (cost at month 0), and the 5 is the slope (cost per additional month).

`8.F.B.5` shifts to qualitative description. Given a verbal story like "a bottle is filled with water, then poured out, then filled again," students sketch a graph showing water-volume versus time. Conversely, given a piecewise graph with rising, flat, falling, and rising segments, students narrate the story. The work supports MP4 (modeling with mathematics) directly.

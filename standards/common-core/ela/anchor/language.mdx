# What are the Common Core Language anchor standards?

<div class="answer-box">
The six Language anchor standards (CCRA.L.1-6) name what every K-12 student should know about the conventions and tools of English. CCRA.L.1-2 cover grammar, usage, and conventions. CCRA.L.3 covers stylistic choice across contexts. CCRA.L.4-6 cover vocabulary: figuring out unfamiliar words, understanding figurative language, and using academic vocabulary. Every grade-level Language (L) standard, kindergarten through grade 12, ladders up to one of these six anchors.

## What it covers

- CCRA.L.1: Use standard English grammar and usage.
- CCRA.L.2: Use standard English capitalization, punctuation, and spelling.
- CCRA.L.3: Make language choices that fit the context.
- CCRA.L.4: Figure out the meaning of unfamiliar words.
- CCRA.L.5: Understand figurative language and word relationships.
- CCRA.L.6: Build and use academic and domain-specific vocabulary.

## Classroom application

Teachers use the Language anchors to plan vocabulary instruction and writing conventions across the year. CCRA.L.1-2 cover grammar and conventions, which usually live in writing conferences and editing checklists. CCRA.L.3 covers stylistic choice, which shows up in revision rubrics. CCRA.L.4-6 cover vocabulary: figuring out new words, understanding figurative language, and using academic vocabulary across subjects. Most state ELA tests embed L.4-6 inside reading items, so vocabulary instruction is often most useful when it sits inside reading and writing lessons rather than as a separate block.
